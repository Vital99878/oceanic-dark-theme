# Oceanic Dark Theme for Brackets

Inspired by Oceanic Dark Theme on Codepen.

## Installation

* Open the Extension Manager in Brackets
* Switch to "Themes" tab
* Search for "Oceanic Dark Theme"
* Click "Install"
